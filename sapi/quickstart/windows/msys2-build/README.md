## 下载msys2

```
.\sapi\quickstart\windows\msys2-build\download-msys2.bat

.\sapi\quickstart\windows\msys2-build\install-msys2.bat

```

## 下载msys2 使用镜像

```
.\sapi\quickstart\windows\msys2-build\download-msys2.bat --mirror china

```

## 进入msys 环境

```

c:\msys64\msys2.exe

```
